# 🌤 Weather Telegram Bot (RU)

Простой и функциональный Telegram-бот для получения текущей погоды в городах России. Проект демонстрирует навыки работы с внешними API, обработки JSON-данных и модульной архитектуры приложения.

## 🚀 Основные функции
* **Поиск по городам РФ:** Автоматическая привязка запросов к российскому региону.
* **Детальный прогноз:** Вывод температуры, влажности, скорости ветра и текстового описания погоды.
* **Обработка ошибок:** Бот корректно реагирует на ввод несуществующих городов или сбои API.
* **Модульная структура:** Код разделен на логические блоки (конфигурация, сервис погоды, логика бота).

## 🛠 Стек технологий
* **Python 3.x**
* **Telebot (pyTelegramBotAPI)** — в текущей версии (планируется переход на **Aiogram 3**)
* **Requests** — для взаимодействия с OpenWeatherMap API
* **Python-dotenv** — для безопасного хранения API-ключей
